# Como colocar no GitHub Pages (gratuito)

## Passo 1 — Criar repositório
1. Vai a github.com e faz login
2. Clica no **+** → **New repository**
3. Nome: `shay-finance`
4. Escolhe **Public**
5. Clica **Create repository**

## Passo 2 — Upload dos ficheiros
1. Na página do repositório, clica **uploading an existing file**
2. Arrasta TODOS os ficheiros desta pasta:
   - `index.html`
   - `beauty-finance.html`
   - `manifest.json`
   - `sw.js`
   - `icon.svg`
   - `icon-192.png`
   - `icon-512.png`
3. (A pasta `db/` com o Excel é opcional — só para teu backup)
4. Escreve "Initial commit" e clica **Commit changes**

## Passo 3 — Ativar GitHub Pages
1. Vai a **Settings** → **Pages** (menu lateral)
2. Em **Source** escolhe **Deploy from a branch**
3. Escolhe **main** e **/ (root)**
4. Clica **Save**
5. Espera 1-2 minutos

## Passo 4 — Aceder e Instalar
1. O teu site fica em: `https://TEU-USERNAME.github.io/shay-finance/`
2. Abre esse link no Chrome do telemóvel
3. O Chrome vai mostrar um banner **"Adicionar ao ecrã inicial"**
4. Ou vai ao menu ⋮ → **Instalar aplicação**
5. Pronto! Aparece no ecrã como uma app com o logo S dourado
