Importar Excel

Importe shay-financas-db.xlsx ou qualquer .xlsx/.csv

Arraste o ficheiro aqui

.xlsx, .xls, .csv

Mapeamento

Os valores são sempre a tua parte (50% salão / 100% extra). O bruto é calculado automaticamente.
Substituir meses: Apaga dados dos meses do ficheiro antes de importar (ideal para reimportar)
Adicionar: Acrescenta sem apagar  |  Limpar TUDO: Reset total

Gestão de Dados
Backup & Segurança

Os teus dados são privados. Ficam apenas neste browser — nunca vão para o GitHub nem para nenhum servidor. O site público só tem o código da app, não os teus valores.

Exportar Backup

Descarrega a base de dados SQLite completa (dados + esquema + categorias)

Restaurar Backup

Carrega um ficheiro .db para restaurar todos os dados

O que fica no backup: Todas as receitas, despesas, categorias, serviços, subcategorias e configurações.
Dica: Faz backup regularmente e guarda numa pasta segura (Google Drive, iCloud, etc.)

Configurações

Prompts da IA

Estes são os textos que definem o comportamento da Shay AI. São enviados em cada mensagem.

Define quem é a IA, o que faz, como se comporta. É enviado em TODAS as mensagens.
Instruções adicionais quando o utilizador envia uma fotografia.
Adicionado ao final do system prompt. Personaliza o comportamento sem alterar o prompt principal.
Chaves API
Selecciona o provider com o radio button, insere a API key, e carrega Guardar. O provider activo fica marcado com EM USO.
Manipulação de Código
Usa um provider dedicado para gerar alterações HTML/CSS (deploy). O Anthropic Claude é recomendado para código.
As chaves ficam apenas no teu browser. Nunca vão para o GitHub.
Esquema da Base de Dados

Este é o esquema que a IA vê em cada mensagem. Podes alterá-lo via chat ("adiciona categoria X").

Ajuda & Comandos

Como Começar

1. Importa o Excel na secção Importar ou adiciona com + Receita / + Despesa

2. Configura a Shay AI no botão ✨ — API keys (Gemini + GitHub)

3. Faz backup em Importar → Backup & Segurança

Comandos da Shay AI

Abre o chat ✨ e diz ou escreve

Registar Receitas
"Fiz microblading no salão, 45 euros, cliente Ana"
"Receita extra, gel, 30 euros"
"Extensão de pestanas no gabinete, 80 euros, cliente Maria"
Registar Despesas
"Gastei 30 euros em marketing no Instagram"
"Despesa de 210 euros, segurança social"
"Comprei material de unhas, 45 euros, MB Way"
Consultas
"Quanto ganhei em Janeiro?"

"Compara receitas salão vs extra"
